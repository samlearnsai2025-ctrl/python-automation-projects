# 🐍 Python Automation Projects

A collection of Python automation tools built to solve real-world problems — extracting data, organizing files, and saving hours of manual work.

---

## 📦 Projects

### 1. 📁 [File Organizer](./file-organizer/)
Automatically sorts messy folders by file type into clean subfolders — with a full undo feature to reverse everything safely.

- `os` `shutil`
- Supports: Images, Documents, Videos, Code, Others
- ✅ Full undo feature

---

### 2. 🌐 [Web Table Extractor](./web-table-extractor/)
Extracts all tables from any public webpage and saves them as CSV files automatically.

- `requests` `pandas`
- Works on Wikipedia, financial sites, sports stats, and more
- ✅ Handles HTTP errors and pages with no tables

---

### 3. 📄 [PDF Table Extractor](./pdf-table-extractor/)
Extracts tables from every page of a PDF file and saves them as clean CSV files.

- `pdfplumber` `pandas`
- Works on reports, invoices, bank statements, and more
- ✅ Skips pages with no tables gracefully
